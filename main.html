CoFRIDA Overview

Offline, we fine-tune a pre-trained Instruct-Pix2Pix model on our self-supervised data. Online, the user can either draw or give the robot a text description to generate a pixel prediction of how the robot should finish the painting. A robot (XArm, Franka, or Sawyer) executes actions, updating the canvas until the user is satisfied.

Self-Supervised Fine-Tuning

Our system allows the robot to enhance existing sketches or generate new content based on the user’s input. Here’s how the simulated sketch evolves over time with robot assistance.
